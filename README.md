## 🚀 Como Usar

### Desenvolvimento Local
1. Abra `index.html` diretamente no navegador
2. Ou use um servidor local:
   ```bash
   python3 -m http.server 8000
   # Acesse http://localhost:8000
   ```

### Publicar no GitHub Pages
1. Crie um repositório no GitHub
2. Faça upload de todos os arquivos
3. Vá em Settings → Pages → escolha a branch `main`
4. Seu site estará disponível em `https://seu-usuario.github.io/portfolio-psicanalista`

## 📝 Personalizações Necessárias

Antes de publicar, atualize os seguintes dados:

### 1. Número de WhatsApp
Procure por `5511999999999` e substitua pelo seu número:
- Header: `https://wa.me/SEUNUMERO`
- Botão flutuante: `href="https://wa.me/SEUNUMERO"`
- Formulário: `https://wa.me/SEUNUMERO?text=...`

### 2. Instagram
Procure por `instagram.com/psicanalista` e substitua pelo seu perfil:
- Header: `https://instagram.com/SEUUSUARIO`
- Footer: `https://instagram.com/SEUUSUARIO`

### 3. Email
Procure por `contato@psicanalista.com` e substitua pelo seu email:
- Footer: `mailto:SEUEMAIL`

### 4. Textos e Conteúdo
- Nome profissional (header, footer)
- Descrição profissional (hero, sobre)
- Credenciais e formação (seção sobre)
- Áreas de atuação (seção atuação)
- Depoimentos (seção prova social)

### 5. Imagens
As imagens já estão integradas via CDN. Se desejar adicionar suas próprias:
- Coloque imagens em `assets/images/`
- Atualize os `src` das imagens no HTML
